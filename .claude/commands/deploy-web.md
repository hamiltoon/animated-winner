description: Deploy the React web app to GitHub Pages
---

Deploy the web app to GitHub Pages by pushing to the main branch.

Steps:
1. Check git status and verify changes
2. Stage all changes with git add
3. Create a commit with a descriptive message
4. Push to main branch
5. Verify GitHub Actions deployment succeeds
6. Check the deployed site at https://hamiltoon.github.io/animated-winner/
